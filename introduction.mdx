DRTC Fluvial Admin is the official administrative panel for the Dirección Regional de Transportes y Comunicaciones (DRTC) of Amazonas, Peru. Built with Next.js 16 and React 19, it provides officials with a secure, modern interface to manage all aspects of fluvial (river) transport regulation — from vessel registration to appointment scheduling and public portal content.

## Key features

- **JWT authentication** with automatic silent token refresh — sessions stay active without interrupting staff workflows
- **Vessel registration (empadronamiento)** — multi-step forms capturing owner data, vessel specs, and motor details with voice-guide accessibility
- **Appointment management** — agenda view, appointment generation, and full appointment history
- **Interactive guided tours** — built-in onboarding tours using driver.js for new staff members
- **Portal web CMS** — manage public tramite listings and news articles displayed on the citizen-facing website
- **Document reception** — receive, review, and track incoming document submissions

## Getting started

<Steps>
  <Step title="Set up your environment">
    Install Node.js 18+ and pnpm, then clone the repository and install dependencies
  </Step>
  <Step title="Configure the backend URL">
    Set the `NEXT_PUBLIC_API_URL` environment variable to point to your NestJS backend
  </Step>
  <Step title="Run the development server">
    Execute `pnpm dev` to start the admin panel at `http://localhost:3000`
  </Step>
  <Step title="Log in with admin credentials">
    Navigate to `/auth` and sign in with your institutional email and password
  </Step>
</Steps>

<Note>
  DRTC Fluvial Admin requires a running NestJS backend. See the [Backend Integration](/configuration/backend-integration) guide to configure the API connection.
</Note>
